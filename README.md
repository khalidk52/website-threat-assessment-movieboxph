## 👨‍💻 Project Overview

This project contains a **security threat assessment of a publicly accessible website (MovieBoxph)** conducted for educational and cybersecurity learning purposes.

The purpose of this analysis is to evaluate the **security posture, domain intelligence, IP information, and potential risks** using passive reconnaissance techniques only.

> ⚠ No hacking, exploitation, or unauthorized access was performed.

---

## 🎯 Objectives

- 🔍 Perform OSINT-based website analysis  
- 🧾 Gather domain registration information  
- 🛡 Check website reputation and threat signals  
- 🌍 Analyze hosting IP and geolocation data  
- 📊 Identify possible security and privacy risks  
- 📄 Document findings in structured report format  

---

## 🧰 Tools Used

### 🌐 OSINT & Threat Intelligence Tools
- 🧾 WHOIS Lookup — Domain registration & ownership details  
- 🛡 VirusTotal — URL reputation & malware detection analysis  
- 🌍 IPInfo — IP address geolocation, ISP, and hosting details  

### 🔍 Supporting Tools
- 🧪 Browser Developer Tools (Network + Console inspection)  
- 🔐 SSL/TLS certificate inspection tools  
- 📡 Manual traffic observation & analysis  

---

## 🔎 Methodology

This assessment was performed using **passive reconnaissance (OSINT techniques)** only.

No exploitation, penetration testing, or unauthorized interaction with the system was performed.

The analysis focused on publicly available data sources to evaluate the website’s security posture.

---

## 📊 Key Findings

- 🧾 Domain WHOIS data is privacy-protected, limiting ownership visibility  
- 🛡 VirusTotal shows mixed reputation signals due to third-party content and ads  
- 🌍 IPInfo indicates the website is hosted behind a CDN or proxy infrastructure  
- ⚠ Heavy use of external scripts may introduce privacy/tracking risks  
- ⚠ Security headers and web hardening need further evaluation  
- 📡 Network analysis shows multiple third-party connections
